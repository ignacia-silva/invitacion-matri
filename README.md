# 💍 Invitación de Matrimonio – Majo & Diego

Página web de invitación para la boda de Majo y Diego, celebrada el **2 de octubre de 2026**.

## Descripción

Invitación digital diseñada con HTML, CSS y JavaScript puro, sin dependencias externas (excepto Google Fonts). El diseño sigue una estética elegante en blanco y negro con acentos en rojo, inspirada en el diseño original creado en Canva.

## Secciones

- **Hero** – Foto principal con nombres y fecha
- **Nos Casamos** – Mensaje de bienvenida
- **Fecha** – Fecha destacada visualmente
- **Ceremonia y Fiesta** – Lugar, día y dirección
- **Itinerario** – Horario del evento
- **Información de la Fiesta** – Dress code, política de adultos y canción
- **Lista de Novios** – Información de regalos y QR
- **Confirmación de Asistencia** – Formulario modal interactivo

## Cómo usar

1. Descarga el archivo `invitacion_majo_diego.html`
2. Abre el archivo en cualquier navegador
3. Para personalizar, edita directamente el HTML:
   - Reemplaza las URLs de las fotos por tus imágenes reales
   - Actualiza los textos con la información del evento
   - Ajusta colores en las variables CSS al inicio del archivo (`:root`)

## Tecnologías

- HTML5 / CSS3 / JavaScript vanilla
- Google Fonts: Cormorant Garamond + Montserrat
- Animaciones con Intersection Observer API

## Personalización rápida

Las variables de color están al inicio del archivo:

```css
:root {
  --rojo: #C0392B;
  --blanco: #F5F0EB;
  --negro: #0A0A0A;
}
```
